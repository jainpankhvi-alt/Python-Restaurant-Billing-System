# 🍽️ Restaurant Billing System

A simple command-line **Restaurant Billing System** built using Python. This project allows users to add food items, calculate the total bill, apply discounts based on the subtotal, and generate a final bill in a clean format.

---

## 📌 Features

* ➕ Add multiple food items
* 💰 Calculate item-wise total
* 🧾 Generate subtotal automatically
* 🎁 Apply discounts based on the total bill
* 📄 Display the final bill with item details
* 🖥️ Simple and user-friendly command-line interface

---

## 🛠️ Technologies Used

* Python
* Functions
* Lists
* Dictionaries
* Loops (`while` & `for`)
* Conditional Statements (`if-elif-else`)
* User Input Handling
